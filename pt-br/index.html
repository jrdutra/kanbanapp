FAQ sobre o KanbanApp

1) Como acessar o KanbanApp sem download?

Você pode abrir o app pelo botão “Acessar app” e utilizar diretamente no navegador, em nova guia. O foco é começar rápido, sem cadastro, para organizar demandas com um quadro kanban de forma prática.

2) Como usar o KanbanApp offline?

Clique no botão de download para baixar o arquivo kanbanapp.html. Depois, abra esse arquivo em qualquer navegador compatível. Assim, você usa o quadro mesmo sem conexão contínua e sem instalação adicional.

3) O KanbanApp armazena dados pessoais?

O app não requer cadastro e não foi construído para guardar dados pessoais em servidor. Os dados do board podem permanecer no armazenamento local do navegador para conveniência de uso.

4) Como fazer backup do meu quadro KANBAN?

Exporte regularmente o board para JSON e mantenha esse arquivo salvo na máquina (ou em local interno aprovado pela sua empresa). Esse hábito protege seu trabalho caso o cache/localStorage seja limpo.

5) Funciona em empresa / ambiente corporativo?

Sim. O KanbanApp pode atender ambientes corporativos por ser leve, estático e permitir operação offline, com menor dependência de serviços externos para o uso básico de quadro KANBAN.

6) Precisa de cadastro?

Não. Você pode abrir e usar imediatamente, sem criar conta. Isso reduz fricção para adoção rápida em uso individual e também em equipes.

Aviso: o KanbanApp não se responsabiliza por perda de dados causada por limpeza de cache do navegador. Para reduzir riscos, exporte seu board em JSON e mantenha cópias de backup no seu computador.
