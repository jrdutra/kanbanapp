看板应用程序常见问题解答

1) 如何在不下载的情况下访问KanbanApp？

您可以使用“打开离线应用程序”按钮打开应用程序，并直接在浏览器的新选项卡中使用它。目标是快速开始， 无需注册，并使用实用的看板来组织需求。

2) 如何离线使用KanbanApp？

点击下载按钮即可获取kanbanapp.html文件。然后在任何兼容的浏览器中打开它。 这使您无需连续连接和额外安装即可使用该板。

3) KanbanApp 是否存储个人数据？

该应用程序不需要注册，也不是为了在服务器上存储个人数据而构建的。棋盘数据可能保留在浏览器的本地存储中 为了使用方便。

4) 如何备份我的看板？

定期将板导出为 JSON 并将文件保存在您的计算机上（或保存在您公司批准的内部位置）。 如果清除缓存/本地存储，这可以保护您的工作。

5）它在企业环境中有效吗？

是的。 KanbanApp可以支持企业环境，因为它是轻量级的、静态的，并且支持离线操作，对外部服务的依赖较低 用于基本看板的使用。

6) 需要注册吗？

不需要。您无需创建帐户即可立即打开并使用它。这减少了个人和团队快速采用的摩擦。

注意：KanbanApp 对浏览器缓存清理造成的数据丢失不承担任何责任。 为了降低风险，请将您的主板导出为 JSON 并在计算机上保留备份副本。
